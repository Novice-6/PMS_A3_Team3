Data Security

All data collected and processed within this inventory management application is used solely for educational purposes as part of the PROG2005 course. All data transmissions follow secure network protocols, and all user inputs are validated to ensure data integrity and security.

User Privacy Protection

We do not collect, store, or share any personal identification information, location data, or device information. All operations performed within the app remain private and are only communicated with the designated course server.

Data Usage and Storage

Inventory data is temporarily stored for functional demonstration only. No commercial use is made of any data entered by users. Sensitive or critical inventory records are protected from accidental deletion or modification.

Application Security Standards

The application implements form validation, input restrictions, and action confirmation dialogs to prevent invalid operations and data loss. All interactive components follow mobile application security best practices.

Educational Use Disclaimer

This application is a student assignment project for demonstration and learning purposes only. The developers assume no liability for data loss, service interruptions, or any issues arising from the use of this application.
